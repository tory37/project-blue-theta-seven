# Release Targets & Input Support

This document outlines the planned platforms and input methods for Blue Theta Seven.

## Platforms

### Primary (Desktop)

- **Steam:**
  - Windows (PC)
  - Linux (including Steam Deck)
  - macOS (Targeted)

### Mobile

- **Android:**
  - Optimized for phones and tablets.

## Input Methods

### Full Controller Support

- Native support for standard gamepads (Xbox, PlayStation, Switch Pro).
- Full navigation of menus and UI via controller.
- Focus on Steam Deck compatibility.

### Touch Screen Support

- Optimized UI for mobile and tablet play.
- Gestures for camera pan, zoom, and tile placement.

### Mouse & Keyboard

- Standard desktop control scheme.
